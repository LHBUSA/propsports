07 Data Sources & Accuracy

PropSports API aggregates data from public sources including the MLB Stats API, NHL Stats API, NBA Stats API, Baseball Savant (Statcast), ESPN, and Open-Meteo (weather). Proprietary Poisson model odds are generated from our own Statcast pipeline — not scraped from sportsbooks. These sources are consistent with those used by major sports analytics platforms.

While we make best efforts to provide accurate, timely data, we make no guarantees of data accuracy, completeness, or timeliness. Sports data is inherently real-time and subject to correction, revision, and delay. Model odds are generated outputs, not official sportsbook lines, and carry their own margin of error.

You should not use PropSports API data as the sole basis for financial decisions. We are not responsible for losses arising from reliance on API data in any context, including sports betting, fantasy sports, or financial trading.

Source APIs may change without notice. We will make reasonable efforts to maintain endpoint compatibility but cannot guarantee uninterrupted access to any specific data field or model output.
