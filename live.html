Under the Hood

Every feature on PropBetEdge.ai
calls this API.

Here's exactly which PropSports API endpoints power each section of the app — so you can see what you'd be getting access to.

/mlb/statcast/batters
HR Pick Engine
Every home run pick starts with Statcast exit velocity, barrel%, hard hit%, and EV spike data from Baseball Savant via PropSports API.
✓ avgEV · barrelPct · hardHitPct · evSpike
/mlb/lineups
Lineup Confirmation
PropBetEdge only surfaces picks for confirmed starters. Every pick card shows batting order position from the official MLB lineups endpoint.
✓ Batting order · Pitcher confirmation
/mlb/weather/all
Park Weather & Factors
Wind speed, direction, temperature, and park factor for every outdoor game. Coors Field HR weather looks very different from Petco Park.
✓ Wind · Temp · Park factor
/mlb/umpires
Umpire Assignments
Home plate umpire data for every game today. K prop models use umpire tendencies as a signal — some umpires dramatically affect strikeout rates.
✓ Home plate ump · Game matchup
/mlb/statcast/pitchers
K Prop Engine
Strikeout picks are built on pitcher swinging strike%, velocity trends, and EV allowed. All sourced from Baseball Savant via PropSports API.
✓ SwStr% · Velo · EV allowed
/mlb/games/live
Live Score Ticker
The live scores ticker at the top of PropBetEdge polls the live games endpoint every 30 seconds during game hours to show real-time scores.
✓ Live scores · Inning · Status
/mlb/schedule/today
Daily Slate
The full day's schedule with probable pitchers, venues, and game times is the foundation every other feature is built on top of.
✓ Pitchers · Times · Venues
/mlb/odds
Best Line Finder
The Best Line section compares moneylines, run lines, and totals from the odds endpoint to surface the sharpest lines for each game.
✓ ML · Run line · O/U
