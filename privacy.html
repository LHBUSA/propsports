02 What We Collect

We collect the minimum data necessary to operate the service:

When you subscribe:

  • Name and email address (used for API key delivery and account communication)
  • Payment information — processed entirely by Stripe. We never see or store your credit card number
  • Your selected plan (Starter, Pro Sport, or Full Bundle)

When you use the API:

  • API key used per request (to authenticate and apply rate limits)
  • Request path and timestamp (stored in Cloudflare KV for rate limiting)
  • Response status codes and latency (logged briefly in Cloudflare Workers for debugging)

When you visit the website:

  • Standard web server logs: IP address, browser type, referring URL, pages visited
  • These logs are retained by Cloudflare and Vercel per their standard policies

We do not collect: location data, device identifiers, behavioral tracking, or any data beyond what is listed above.

03 How We Use Your Data

  • API key delivery — your email is used to send your API key immediately after purchase via Resend
  • Account communication — we may email you about service changes, outages, or subscription status
  • Rate limiting — your API key is stored in Cloudflare KV to enforce per-plan request limits
  • Billing — Stripe uses your payment information to process subscriptions and send receipts
  • Internal notifications — new subscriptions trigger an internal Slack notification for operational awareness
  • Improving the service — aggregate (non-identifiable) usage patterns help us understand which endpoints are most used

We do not use your data for advertising, profiling, or sale to third parties.

04 Who We Share It With

We share data only with service providers necessary to operate PropSports API:

  • Stripe — payment processing. stripe.com/privacy
  • Cloudflare — API hosting, edge caching, KV storage. cloudflare.com/privacypolicy
  • Vercel — website hosting. vercel.com/legal/privacy-policy
  • Resend — transactional email delivery for API key emails. resend.com/legal/privacy-policy

We do not sell, rent, or trade your personal information to any third party. Ever.

We may disclose information if required to do so by law or in the good-faith belief that such action is necessary to comply with a legal obligation.

05 Storage & Security

Your data is stored in the following locations:

  • API keys and rate limit counters — Cloudflare KV (encrypted at rest, globally distributed)
  • Subscription and payment records — Stripe's secure infrastructure
  • Email records — Resend's secure transactional email infrastructure

We do not operate our own database storing personal information beyond what is held by these providers. API request logs on Cloudflare Workers are ephemeral and not persisted beyond standard log retention.

While no system is 100% secure, we use industry-standard practices: HTTPS everywhere, API key authentication, rate limiting, and minimal data collection.

06 Cookies & Tracking

The PropSports API website (propsports.proptechusa.ai) does not use analytics cookies, advertising cookies, or third-party tracking scripts.

Stripe's checkout flow uses cookies necessary for payment processing when you click a pricing plan button. These are limited to the Stripe checkout session.

Cloudflare may set a security cookie (__cf_bm) for bot management purposes. This is a functional security measure, not a tracking cookie.

We do not run Google Analytics, Meta Pixel, or any behavioral tracking system on this site.

07 Your Rights

Depending on your location, you may have the following rights regarding your personal data:

  • Access — request a copy of the personal data we hold about you
  • Correction — request correction of inaccurate data
  • Deletion — request deletion of your data. We will delete your email and API key records upon cancellation and request

  • Objection — object to processing of your data in certain circumstances

To exercise any of these rights, contact us at 1-888-784-3881. We will respond within 30 days.

To cancel your subscription and stop all data processing, you can cancel directly through Stripe or contact us.

08 Children

PropSports API is not directed at children under the age of 13. We do not knowingly collect personal information from children under 13. If you believe we have inadvertently collected such information, please contact us immediately and we will delete it.
