We don't scrape odds.
We generate them.

Every other sports API depends on DraftKings, FanDuel, or a third-party odds feed. We built a Poisson probability model on top of our owned Statcast data and generate fair-value odds 6× a day — so when sportsbooks block scrapers, you stay up.

📊

Owned Statcast pipeline

Exit velocity, barrel%, xwOBA, pitch mix — aggregated nightly into our own Supabase. No vendor dependency.

🧠

Poisson distribution model

Each player's expected stat output modeled against opposing pitcher, park factor, and recent splits. True probability → American odds.

🔄

Refreshed 6× daily

8AM, 10AM, 12PM, 2PM, 5PM, 8PM CT via Cloudflare cron. Reflects late lineup changes, weather shifts, pitcher swaps.
